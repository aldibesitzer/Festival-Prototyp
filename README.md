# Paléo Festival 2026 — Landingpage Prototyp

Interaktiver Prototyp für die Paléo Festival 2026 Landingpage.

## Inhalt

- **Home** — Hero mit Live-Countdown und Call-to-Actions
- **Line-up** — Vier Festivaltage mit hervorgehobenen Headlinern und Artist-Preview-Hover
- **Tickets** — Drei Ticketoptionen
- **Warenkorb** — Funktionierende Bestellübersicht

## Features

- Live-Countdown bis zum Festivalstart (22.07.2026)
- Hover-Animationen auf Navigation, Buttons, Tickets und Lineup-Cards
- Artist-Preview beim Hover über Lineup-Cards
- Sticky Navigation mit Blur-Effekt beim Scrollen
- Animiertes Sternenfeld passend zum «Kosmos»-Thema
- Responsive (Desktop & Mobile)
